### To install prisma run 

```bash
npm install prisma @prisma/client
```

### After the installation, we can then initialize Prisma with

```bash
npx prisma init
```


### RUN THIS To generate prisma after changing model 

```bash
npx prisma generate
```

### After creating new model run
```bash
npx prisma db push
```
